# Overwatch Scala Game (GUI)
This GUI application is a turn-based battle game inspired by the game â€œOverwatch" from Blizzard Entertainment. The game is a two-player battle game where each player picks a character called Heroes from one of the three roles to defeat their opponent. The roles are Tank, Damage, and Support. Each role gives the hero a unique passive trait. Besides that, each hero has a unique set of abilities which can carried out different behaviours such as attack, heal, or boost agility.  

The game was developed using Scala and object-oriented programming concepts.
